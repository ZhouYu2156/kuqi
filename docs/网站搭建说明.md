# 秘术云阁品牌站 · 搭建与维护说明

> 对应产品需求见仓库根目录 `PR.md`。本文档说明当前代码结构、页面与模块、内容维护方式及后续可扩展点。

---

## 1. 技术栈与运行方式

| 项目 | 说明 |
|------|------|
| 框架 | Nuxt 4 |
| UI | Nuxt UI 4（布局、导航、卡片、Hero 等） |
| 表单 | Element Plus（联系页） |
| 样式 | Tailwind CSS 4（`app/assets/css/main.css`）+ Element 变量（`app/assets/element/index.scss`） |
| 内容 | `@nuxt/content` 博客集合 `blog`，源文件为 `content/blog/*.md` |
| 主题 | `@nuxt/ui` 内置 Color Mode：默认深色，可用顶栏按钮切换亮/暗 |

**本地开发**

```bash
pnpm install
pnpm dev
```

默认开发地址：`http://127.0.0.1:3000`（见 `nuxt.config.ts` 中 `devServer`）。

**生产构建**

```bash
pnpm build
pnpm preview
```

  pages/
    index.vue             # 首页：价值主张、三条主线、定价摘要、私域区块
    services.vue          # 服务与定价（199 / 599 + 全栈交付说明）
    contact.vue           # 联系表单 + SiteWechatBlock

## 4. 设计实现要点

- **版心与留白**：通用类 `.site-container`（`max-w-6xl` + 水平 padding），与各页 `UPageHeader` 搭配使用。
- **黑金主色（与 `personal_brand_website_plan.html` 对齐）**：主色金 `#C9A84C`（`--gold`）、浅金 `#E8C97A`、暗金 `#8A6D2F`；背景 `#0A0A0A` / 卡片 `#161616`；正文与弱化字 `#F0EBE0` / `#8A8070`；描边 `rgba(201,168,76,0.25)`。Nuxt UI `primary` 映射为 Tailwind `brand`；Element Plus 主色同步为 `#C9A84C`。
- **字体**：标题展示用 **Playfair Display**（`font-display`），正文 **Noto Serif SC**，标签与页脚说明用 **JetBrains Mono**（`font-mono-site`）。
- **响应式**：Nuxt UI 组件与栅格（`UPageGrid` 等）在移动端堆叠，顶栏使用 `slideover` 模式。
- **品牌 Logo**：`public/logo-mishu.svg`（阁顶 + 云线 + 金渐变，与顶栏一致）；`nuxt.config` 中 favicon 指向该文件。
- **微信 / 公众号二维码**：实拍图 `public/wechat-qrcode.png`、`public/wechat-official-account-qrcode.png`，由 `site/SiteQrPair.vue` 引用。**页脚不再重复展示二维码**，仅在首页「微信与公众号」区块与联系页展示。
- **主 CTA**：首页/服务页关键按钮使用 `btn-gold-solid`（金渐变底 + `#0a0a0a` 字），避免与背景糊成一片。

## 5. 博客发布流程

1. 在 `content/blog/` 新建 `xxx.md`（文件名即 URL 中的 `slug`）。
2. 顶部 **YAML frontmatter** 必填建议字段：

```yaml
---
title: 文章标题
description: 摘要
date: '2026-03-31'
---
```

3. 正文使用 Markdown；详情页通过 `<ContentRenderer :value="post" />` 渲染。
4. 列表按 `date` **降序** 排列（见 `blog/index.vue` 中 `order('date', 'DESC')`）。

## 6. 联系表单与接口

- 前端：`POST /api/contact`，请求体字段 `name`, `contact`, `intent`, `budget`, `message`。
- 服务端：`server/api/contact.post.ts` 做必填校验，成功返回 `{ ok: true, message: '...' }`。
- **当前未接邮件/数据库**，生产环境请改为：写库、发邮件、或对接飞书/企业微信机器人等。
